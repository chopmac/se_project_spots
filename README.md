# Spots

## Description
Spots is a responsive social media platform where users can share and discover beautiful locations. The website features a clean, modern design that adapts seamlessly across desktop, tablet, and mobile devices.

## Technologies Used
- HTML5
- CSS3
- BEM methodology
- Responsive design
- Flexbox/Grid
- Google Fonts (Poppins)
